# 🚀 Run Locally

Clone repository:

```bash
git clone https://github.com/<your-username>/java-interview-playbook.git
```

Go to folder:

```bash
cd pattern-print
```

Compile:

```bash
javac FileName.java
```

Run:

```bash
java FileName
```

Example:

```bash
javac DiamondPattern.java
java DiamondPattern
```
